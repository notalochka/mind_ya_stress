# Mind Я


## Технології

- Next.js 15
- React 19
- TypeScript
- CSS Modules

## Встановлення

```bash
npm install
```

## Запуск проекту

```bash
npm run dev
```

Відкрийте [http://localhost:3000](http://localhost:3000) у браузері.

## Структура проекту

```
src/
  components/     # React компоненти
  pages/          # Сторінки Next.js
  styles/         # Глобальні стилі
  types/          # TypeScript типи
  lib/            # Утиліти та хуки
```

## Доступні сторінки

- `/` - Головна сторінка
